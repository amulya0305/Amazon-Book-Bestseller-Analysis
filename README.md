# Amazon-Book-Bestseller-Analysis
A Python data analysis project using the Pandas library to explore Amazon bestselling books. The project imports data from a CSV file taken from kaggle and analyzes genres, authors, ratings, reviews, prices, and yearly bestseller trends.

# Amazon Bestseller Books Analysis

This project analyzes Amazon bestselling books using Python and the Pandas library. It imports data from a CSV file and performs exploratory data analysis (EDA) to generate meaningful insights about bestselling books.

## Features
- Import data from a CSV file using Pandas
- Display dataset information
- Check for missing and duplicate values
- Analyze Fiction vs Non-Fiction books
- Find top authors
- Identify highest-rated and most-reviewed books
- Calculate average ratings and prices
- Analyze yearly bestseller trends

## Technologies Used
- Python 3
- Pandas

## Dataset
Amazon Bestselling Books Dataset (CSV)
